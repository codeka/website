War Worlds

War Worlds is Android-based space strategy multi-player game. The game is set in an infinitely-large universe where players occupy their own region of stars. Players interact via in-game chat, form alliances and wage war on an epic scale.

As of April 2014, War Worlds has around 1,500 7-day active users and is continuing to grow.

The technology behind War Worlds is varied and complex. The server is a Java application, utilizing Jetty for an HTTP interface. Nginx is used as the front-end to provide a secure SSL interface to the game client. The game client is built using a combination of Android technologies, including andengine, a 2D OpenGL ES library for improved graphics performance.

The Motor Report

The Motor Report is a leading Australian motoring news and review site. The site is maintained by a team of writers and editors, and has been in operating since 2007.

Pic Scan

Pic Scan is an open source app that I developed which allows you to take pictures of documents with your phone, and it will automatically flatten and filter the image so that it appears like your scanned it with a flatbed scanner.

I built this app because I wasn't particularly happy with the closed-source alternatives (CamScanner, one of more popular closed-source alternatives is so riddled with ads, and it puts a watermark on every document you scan, unless you pay some insane monthly fee for the 'premium' version.)

The algorithms are actually not that complicated, and open source libraries like OpenCV made it even easier.

Extreme Activties

ExtremeActivties.com.au is a website dedicated to linking you to extreme and other outdoor activites that the whole family can enjoy. With links to V8 and Rally driving, paintball, Jet Fighter flights, Skiing, Snowboard, White Water rafting and more, there’s sure to be something to excite you!

The site was built by Codeka Consulting from the ground up. The goal was a simple-to-use layout that presented the information to the user in an easy to read and easy to understand format.

The site partnered with godo.com.au who provides the actual activities and an API between the two sites keeps the Extreme Activities site constantly up-to-date with all of the latest activities available there.
